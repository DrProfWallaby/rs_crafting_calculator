# rs_crafting_calculator
A crafting calculator for RuneScape 3

## What it does
This is a relatively simple CLI calculator where you type in the name of the item you want to make and how many you want to make and it tells you all of the items you need to craft to craft the intial item.

## How to use it
From the project folder run source bin/activate, then run python ./src/main.py. It asks you what you want to make, and how many you want to make and that's it!

## Requirements
I think Beautiful Soup is the only package that needs to be installed directly. You'll obviously need to install Python as well. I didn't make more notes on that because I wasn't planning on properly distributing this program.

## Notes
Necromancy recipes were proving to be a little too complicated so they aren't supported at the moment.
Searching is not implemented, so you'll need to know what item you want to make ahead of time.
You also need to type the roman numerals for things like Signs of the Porter (IE Sign of the porter II not Sign of the porter 2)
